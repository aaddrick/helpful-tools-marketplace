# Helpful Tools Marketplace

A curated marketplace of helpful Claude Code plugins for developers.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add aaddrick/helpful-tools-marketplace
```

## Available Plugins

### gh-cli-search

Comprehensive GitHub CLI search commands and troubleshooting.

**Features:**
- Search code across repositories
- Search commits, issues, and pull requests
- Advanced GitHub CLI search syntax
- Troubleshooting helpers

**Install:**
```bash
/plugin install gh-cli-search@helpful-tools
```

**Repository:** [aaddrick/gh-cli-search](https://github.com/aaddrick/gh-cli-search)
